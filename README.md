# Cofrinho digital

App made with Flutter.

## Getting started

Clone this repo.

```bash
git clone https://github.com/jpssantiago/cofrinho_digital
```

Install dependencies

```bash
flutter pub get
```

Enjoy :)

```bash
flutter run
```
